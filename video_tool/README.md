# 動画ツール

動画の取り込み・再生／一時停止・コマ送り・キャプチャ・トリミングができる GUI ツールです。

## 起動

ルートの `python run.py` から「動画ツール」を選ぶか、このフォルダで `python gui.py` を実行。

## セットアップ

```bash
pip install -r requirements.txt
```

## 機能

- **動画を開く** … MP4 / AVI / MOV / MKV 等を開く
- **再生 / 一時停止 / 停止** … 再生コントロール
- **シーク** … スライダーで任意位置へ
- **コマ送り** … 種類で「次へ 1/5/10/30 コマ」「前へ 1/5/10/30 コマ」を選択して実行
- **キャプチャ** … 現在のフレームを PNG/JPEG で保存
- **トリミング** … 開始・終了フレームを指定して範囲を切り出し、別ファイルに保存

## トリミングのやり方

**注意**: トリミングは**再生を一時停止したときだけ**使えます（再生中は欄がグレーアウトします）。

1. **動画を開く** … 「動画を開く」でファイルを選択
2. **一時停止する** … 再生中なら「一時停止」を押す
3. **開始・終了を決める**（どれか一つ、または組み合わせでOK）
   - **スライダー**: 「開始」「終了」のスライダーをドラッグまたはクリックしてフレームを指定
   - **現在位置を使う**: 上のシークスライダーやコマ送りで位置を合わせて「ここを開始に」「ここを終了に」を押す
   - **数値で指定**: 「開始」「終了」の数値欄にフレーム番号を直接入力
4. **「トリミングして保存」** … 保存先とファイル名を選ぶと、その範囲だけが新しい動画ファイルとして保存されます
